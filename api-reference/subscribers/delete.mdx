---
title: 'Delete subscriber'
openapi: 'DELETE /subscribers/{id}'
---

Remove a subscriber from a post. The user will no longer receive notifications about post updates.

## Path parameters

<ParamField path="id" type="string" required>
  UUID of the subscription to delete
</ParamField>

## Response

<ResponseField name="deletedSubscriber" type="object">
  The deleted subscriber object
</ResponseField>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://app.tryordinal.com/api/v1/subscribers/333e4444-e89b-12d3-a456-426614174008 \
  --header 'Authorization: Bearer YOUR_API_KEY'
```
