---
title: 'List all labels'
openapi: 'GET /labels'
---

List all labels in your workspace. Labels help organize and categorize posts.

## Response

<ResponseField name="labels" type="array">
  Array of label objects
  
  <Expandable title="label object">
    <ResponseField name="id" type="string">
      UUID of the label
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Label name
    </ResponseField>
    
    <ResponseField name="color" type="string">
      Text color (hex code)
    </ResponseField>
    
    <ResponseField name="backgroundColor" type="string">
      Background color (hex code)
    </ResponseField>
    
    <ResponseField name="createdAt" type="datetime">
      Timestamp when the label was created
    </ResponseField>
    
    <ResponseField name="updatedAt" type="datetime">
      Timestamp when the label was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://app.tryordinal.com/api/v1/labels \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch(
  'https://app.tryordinal.com/api/v1/labels',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://app.tryordinal.com/api/v1/labels',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
```
