## Overview

Labels help you categorize and filter your posts. Create custom labels to match your workflow and keep your content organized.

## Creating Labels

1. Go to **Labels** â†’ **New Label**
2. Enter label name
3. Choose a color
4. Add description (optional)
5. Save your label

## Applying Labels

### To Individual Posts

Apply labels while creating or editing posts:
- Click the label icon in the composer
- Select one or more labels
- Labels appear on the post card

### Bulk Labeling

Apply labels to multiple posts:
1. Select posts in list view
2. Click **Add Labels**
3. Choose labels to apply
4. Confirm changes

## Using Labels

### Filtering

Filter your content by label:
- In Posts list view
- On the Calendar
- In Campaign views

### Reporting

Track performance by label:
- See aggregate metrics per label
- Compare label performance
- Identify top-performing categories

## Managing Labels

### Edit Labels

Update label names, colors, or descriptions at any time. Changes apply to all labeled posts.

### Delete Labels

Remove labels you no longer need. Posts will be un-labeled but not deleted.
