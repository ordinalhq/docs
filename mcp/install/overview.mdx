## Before you start

You'll need an Ordinal account with access to at least one workspace. No API keys required — the MCP server signs you in through your browser the first time you use it.

<Info>
  **Using another MCP client?** If your client supports the Streamable HTTP transport with OAuth, point it at `https://app.tryordinal.com/mcp` and follow its own MCP setup docs. Reach out to [support@tryordinal.com](mailto:support@tryordinal.com) if you get stuck.
</Info>

## Already have the old MCP server installed?

If you previously set up the server at `https://app.tryordinal.com/api/mcp` with a workspace API key, remove it first. Running both at the same time confuses the assistant — it sees duplicate tools.
