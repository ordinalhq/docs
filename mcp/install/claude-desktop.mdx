---
title: "Claude Desktop"
description: "Add the Ordinal MCP server to Claude Desktop"
---

<Steps>
  <Step title="Open Claude Desktop settings">
    Press `Cmd + ,` on Mac or `Ctrl + ,` on Windows.
  </Step>
  <Step title="Go to Connectors">
    In the sidebar, click **Connectors**, then click **Add custom connector**.
  </Step>
  <Step title="Paste the server URL">
    Use the Ordinal MCP server URL:

    ```
    https://app.tryordinal.com/mcp
    ```

    Give it a name like `Ordinal` and save.
  </Step>
  <Step title="Sign in with Ordinal">
    Claude Desktop opens a browser window. Sign in with your Ordinal account and approve access. The connector should now show **Connected**.
  </Step>
  <Step title="Try it out">
    In a chat, ask: *"List my Ordinal workspaces."* Claude should call the `ordinal_get_workspace_context` tool and return your workspaces.
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="The connector won't sign in">
    Make sure you're on the latest version of Claude Desktop — older versions don't support OAuth-based MCP connectors. Restart the app after updating.
  </Accordion>
  <Accordion title="I don't see Ordinal tools in the chat">
    Open the connector panel in chat (the icon in the message bar) and confirm Ordinal is enabled. If it says **Reconnect**, click it to re-run the sign-in.
  </Accordion>
  <Accordion title="I'm on an older build without custom connectors">
    If your Claude Desktop version doesn't have the **Add custom connector** button, follow the [Claude Code](/mcp/install/claude-code) instructions and use the Claude Code CLI instead, or upgrade Claude Desktop to a recent build.
  </Accordion>
</AccordionGroup>
