## Overview

Manage your personal Ordinal account settings, preferences, and profile information.

## Profile Information

### Update Your Profile

1. Click your avatar in the top right
2. Select **Profile Settings**
3. Update your information
4. Save changes

### Profile Fields

- **Name** - Your display name
- **Email** - Login email (verified)
- **Avatar** - Profile picture
- **Timezone** - Your local timezone
- **Language** - Interface language

## Security

### Change Password

1. Go to **Profile Settings** → **Security**
2. Click **Change Password**
3. Enter current password
4. Enter new password twice
5. Save changes

### Two-Factor Authentication

Enable 2FA for additional security:

1. Go to **Profile Settings** → **Security**
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

### Active Sessions

View and manage your login sessions:
- See all active sessions
- Device and location info
- Revoke sessions if needed

## Preferences

### Interface Preferences

Customize your Ordinal experience:
- **Theme** - Light, dark, or system
- **Compact mode** - Denser UI layout
- **Default view** - Calendar, list, or board

### Default Settings

Set defaults for new content:
- Default profile selection
- Default labels
- Auto-save preferences

### Social Logins

Manage connected login methods:

### Personal Integrations

Connect personal tools:
- Calendar sync
