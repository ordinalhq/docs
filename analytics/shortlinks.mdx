## Overview

Shortlinks make your URLs cleaner and trackable. Create branded short links for any URL and track clicks, conversions, and engagement.

## Creating Shortlinks

### Quick Create

1. Click **New Shortlink** or paste a URL
2. Enter the destination URL
3. Customize the short code (optional)
4. Add to a post or copy the link

### In Post Composer

Create shortlinks while drafting:
1. Paste a URL in your post
2. Click "Shorten" on the URL
3. Link is automatically shortened
4. Tracking is enabled

## Shortlink Features

### Custom Slugs

Create memorable short links:
- `ord.co/product-launch`
- `ord.co/summer-sale`
- `ord.co/webinar-signup`

### Custom Domains

Use your own domain:
- Set up a custom domain in settings
- All links use your brand
- Example: `links.yourbrand.com/offer`

### UTM Parameters

Automatically add tracking parameters:
- Campaign source
- Medium
- Campaign name
- Custom parameters

## Analytics

### Click Tracking

See who's clicking your links:
- Total clicks over time
- Unique vs. repeat visitors
- Geographic distribution

### Device & Browser

Understand your audience:
- Mobile vs. desktop
- Browser breakdown
- Operating systems

### Referrers

See where clicks come from:
- Social platform
- Direct visits
- Other websites

## Managing Shortlinks

### Edit Links

Update shortlink settings:
- Change destination URL
- Update UTM parameters
- Modify expiration

### Archive Links

Remove unused links from active view:
- Links still work
- Hidden from main list
- Accessible in archive
