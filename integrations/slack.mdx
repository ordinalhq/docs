## Overview

Integrate Slack with Ordinal to receive notifications, coordinate approvals, and keep your team in sync with your social media activity.

## Connecting Slack

1. Go to **Settings** → **Integrations**
2. Find **Slack** and click **Connect**
3. Select your Slack workspace
4. Choose channels Ordinal can access
5. Authorize the connection

## Features

### Notifications

Receive Ordinal notifications in Slack:
- Post published notifications
- Approval requests
- Comment mentions
- Failed post alerts

### Approval Workflows

Handle approvals directly in Slack:
- Receive approval requests as messages
- Approve or reject with buttons
- Add feedback in thread

### Team Coordination

Coordinate content activities:
- Boost channel notifications
- Content calendar updates
- Performance alerts

## Configuration

### Channel Mapping

Set up channels for different notification types:

| Notification Type | Suggested Channel |
|------------------|-------------------|
| All notifications | #social-media |
| Approvals | #content-approvals |
| Alerts | #social-alerts |
| Boosts | #team-boosts |

### Notification Preferences

Customize what gets sent to Slack:
1. Go to **Settings** → **Integrations** → **Slack**
2. Select notification types
3. Choose destination channels
4. Save preferences

## Slack Commands

Use slash commands in Slack:
- `/ordinal status` - View scheduled posts
- `/ordinal approve` - See pending approvals
- `/ordinal analytics` - Quick metrics summary
