---
title: "post.inline_comment.created"
description: "Webhook event fired when an inline, text-anchored comment is added to an Ordinal post. Sends one event per comment, including replies inside an existing thread."
sidebarTitle: "post.inline_comment.created"
---

All webhook payloads follow a consistent top-level structure with event-specific data nested within the `data` object. Inline comments attach to specific text in a post—this event fires for each new comment or reply in a thread.

<ResponseField name="type" type="string" required>
  The event type (e.g. `post.inline_comment.created`).
</ResponseField>

<ResponseField name="data" type="object" required>
  Event-specific payload for this webhook.

  <Expandable title="Data object parameters">
    <ResponseField name="comment" type="object" required>
      The inline comment.

      <Expandable title="Comment object parameters">
        <ResponseField name="thread" type="object" required>
          The inline comment thread. Contains `id`, `channel` (channel the highlighted text is in), `highlightedText` (the anchored text), `resolved`, `createdAt`, and `updatedAt`.
        </ResponseField>

        <ResponseField name="id" type="string" required>
          Comment ID.
        </ResponseField>

        <ResponseField name="message" type="string" required>
          Comment message text.
        </ResponseField>

        <ResponseField name="post" type="object" required>
          Post the comment was added to. Contains `id`, `title`, and `url`.
        </ResponseField>

        <ResponseField name="workspace" type="object" required>
          Workspace the post belongs to. Contains `id`, `slug`, and `name`.
        </ResponseField>

        <ResponseField name="createdBy" type="object" required>
          User who added the comment. Contains `id`, `firstName`, `lastName`, and `email`.
        </ResponseField>

        <ResponseField name="createdAt" type="string" required>
          ISO 8601 timestamp when the comment was created.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp when the webhook event was created.
</ResponseField>
