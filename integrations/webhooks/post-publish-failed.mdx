---
title: "post.publish_failed"
description: "Fired when a post fails to publish"
sidebarTitle: "post.publish_failed"
---

All webhook payloads follow a consistent top-level structure with event-specific data nested within the `data` object.

<ResponseField name="type" type="string" required>
  The event type (e.g. `post.publish_failed`).
</ResponseField>

<ResponseField name="data" type="object" required>
  Event-specific payload for this webhook.

  <Expandable title="Data object parameters">
    <ResponseField name="id" type="string" required>
      Post ID.
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Post title.
    </ResponseField>

    <ResponseField name="channel" type="string" required>
      Channel where publish failed.
    </ResponseField>

    <ResponseField name="error" type="string" required>
      Error message describing the failure.
    </ResponseField>

    <ResponseField name="campaign" type="object | null">
      Campaign if assigned. Contains `id` and `name`.
    </ResponseField>

    <ResponseField name="url" type="string" required>
      Ordinal post URL.
    </ResponseField>

    <ResponseField name="profile" type="object" required>
      Profile that attempted to publish. Contains `id`, `name`, and `detail`.
    </ResponseField>

    <ResponseField name="scheduledPublishAt" type="string" required>
      ISO 8601 timestamp when publish was attempted.
    </ResponseField>

    <ResponseField name="workspace" type="object" required>
      Workspace the post belongs to. Contains `id`, `slug`, and `name`.
    </ResponseField>

    <ResponseField name="createdBy" type="object" required>
      User who created the post. Contains `id`, `firstName`, `lastName`, and `email`.
    </ResponseField>

    <ResponseField name="failedAt" type="string" required>
      ISO 8601 timestamp when the publish failed.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp when the webhook event was created.
</ResponseField>
