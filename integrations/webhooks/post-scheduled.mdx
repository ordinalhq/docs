---
title: "post.scheduled"
description: "Fired when a post is scheduled for publishing"
sidebarTitle: "post.scheduled"
---

All webhook payloads follow a consistent top-level structure with event-specific data nested within the `data` object.

<ResponseField name="type" type="string" required>
  The event type (e.g. `post.scheduled`).
</ResponseField>

<ResponseField name="data" type="object" required>
  Event-specific payload for this webhook.

  <Expandable title="Data object parameters">
    <ResponseField name="id" type="string" required>
      Post ID.
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Post title.
    </ResponseField>

    <ResponseField name="channels" type="array" required>
      Channels the post is scheduled for.
    </ResponseField>

    <ResponseField name="campaign" type="object | null">
      Campaign if assigned. Contains `id` and `name`.
    </ResponseField>

    <ResponseField name="url" type="string" required>
      Ordinal post URL.
    </ResponseField>

    <ResponseField name="publishAt" type="string" required>
      ISO 8601 timestamp when the post will publish.
    </ResponseField>

    <ResponseField name="postedImmediately" type="boolean" required>
      Whether the post was published immediately.
    </ResponseField>

    <ResponseField name="workspace" type="object" required>
      Workspace the post belongs to. Contains `id`, `slug`, and `name`.
    </ResponseField>

    <ResponseField name="scheduledBy" type="object" required>
      User who scheduled the post. Contains `id`, `firstName`, `lastName`, and `email`.
    </ResponseField>

    <ResponseField name="scheduledAt" type="string" required>
      ISO 8601 timestamp when the post was scheduled.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp when the webhook event was created.
</ResponseField>

<ResponseExample>

```json Example
{
  "type": "post.scheduled",
  "data": {
    "id": "clx123abc",
    "title": "Q1 Launch",
    "channels": ["LinkedIn", "Twitter"],
    "campaign": { "id": "clx456", "name": "Launch 2025" },
    "url": "https://app.tryordinal.com/my-workspace/posts/clx123abc",
    "publishAt": "2025-03-01T14:00:00.000Z",
    "postedImmediately": false,
    "workspace": { "id": "ws_1", "slug": "my-workspace", "name": "My Workspace" },

    "scheduledAt": "2025-02-26T12:00:00.000Z"
  },
  "createdAt": "2025-02-26T12:00:00.000Z"
}
```

</ResponseExample>
