---
title: "post.rescheduled"
description: "Fired when a post's scheduled time is changed"
sidebarTitle: "post.rescheduled"
---

All webhook payloads follow a consistent top-level structure with event-specific data nested within the `data` object.

<ResponseField name="type" type="string" required>
  The event type (e.g. `post.rescheduled`).
</ResponseField>

<ResponseField name="data" type="object" required>
  Event-specific payload for this webhook.

  <Expandable title="Data object parameters">
    <ResponseField name="id" type="string" required>
      Post ID.
    </ResponseField>

    <ResponseField name="title" type="string" required>
      Post title.
    </ResponseField>

    <ResponseField name="channels" type="array" required>
      Channels the post is scheduled for.
    </ResponseField>

    <ResponseField name="campaign" type="object | null">
      Campaign if assigned. Contains `id` and `name`.
    </ResponseField>

    <ResponseField name="url" type="string" required>
      Ordinal post URL.
    </ResponseField>

    <ResponseField name="previousPublishAt" type="string" required>
      ISO 8601 timestamp of the previous publish time.
    </ResponseField>

    <ResponseField name="newPublishAt" type="string" required>
      ISO 8601 timestamp of the new publish time.
    </ResponseField>

    <ResponseField name="workspace" type="object" required>
      Workspace the post belongs to. Contains `id`, `slug`, and `name`.
    </ResponseField>

    <ResponseField name="rescheduledBy" type="object" required>
      User who rescheduled the post. Contains `id`, `firstName`, `lastName`, and `email`.
    </ResponseField>

    <ResponseField name="rescheduledAt" type="string" required>
      ISO 8601 timestamp when the post was rescheduled.
    </ResponseField>

    <ResponseField name="linkedIn" type="object | null">
      LinkedIn content when the post includes LinkedIn. Contains `profile` (id, name, detail), `copy`, and `assets` (id, name, url, mimetype, size, width, height, duration).
    </ResponseField>

    <ResponseField name="x" type="object | null">
      X/Twitter content when the post includes Twitter. Contains `profile` (id, name, detail) and `tweets` (array of `{ copy, assets }`).
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp when the webhook event was created.
</ResponseField>
