---
title: "social_profile.connected"
description: "Webhook event fired when a social profile is connected to an Ordinal workspace, including the platform, profile identifier, and connecting user for audit logs."
sidebarTitle: "social_profile.connected"
---

All webhook payloads follow a consistent top-level structure with event-specific data nested within the `data` object.

<ResponseField name="type" type="string" required>
  The event type (e.g. `social_profile.connected`).
</ResponseField>

<ResponseField name="data" type="object" required>
  Event-specific payload for this webhook.

  <Expandable title="Data object parameters">
    <ResponseField name="profile" type="object" required>
      The connected profile.

      <Expandable title="Profile object parameters">
        <ResponseField name="id" type="string" required>
          Profile ID.
        </ResponseField>

        <ResponseField name="name" type="string | null">
          Profile display name.
        </ResponseField>

        <ResponseField name="detail" type="string | null">
          Profile vanity name or handle.
        </ResponseField>

        <ResponseField name="channel" type="string" required>
          Social channel (e.g. LinkedIn, Twitter).
        </ResponseField>

        <ResponseField name="profileImageUrl" type="string | null">
          URL of the profile image.
        </ResponseField>

        <ResponseField name="type" type="string">
          Optional. LinkedIn account type (e.g. Company, Personal).
        </ResponseField>

        <ResponseField name="url" type="string">
          Optional. Profile URL on the social platform.
        </ResponseField>

        <ResponseField name="workspace" type="object" required>
          Workspace the profile was connected to. Contains `id`, `slug`, and `name`.
        </ResponseField>

        <ResponseField name="connectedBy" type="object" required>
          User who connected the profile. Contains `id`, `firstName`, `lastName`, and `email`.
        </ResponseField>

        <ResponseField name="connectedAt" type="string" required>
          ISO 8601 timestamp when the profile was connected.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp when the webhook event was created.
</ResponseField>
