## Overview

Integrate Dub.co with Ordinal for advanced link management, custom domains, and enhanced link analytics.

## What is Dub.co?

Dub.co is a modern link management platform that provides:
- Custom branded short links
- Advanced analytics
- Link management at scale
- API-first architecture

## Connecting Dub.co

<Steps>
  <Step title="Get Your API Key">
    Log in to your Dub.co account and navigate to API settings to copy your API key.
  </Step>
  <Step title="Open Integrations">
    In Ordinal, go to **Settings** → **Integrations**.
  </Step>
  <Step title="Find Dub.co">
    Locate **Dub.co** and click **Connect**.
  </Step>
  <Step title="Enter API Key">
    Paste your Dub.co API key.
  </Step>
  <Step title="Select Workspace">
    Choose your default Dub.co workspace.
    
    <Check>
    Dub.co shows as "Connected" in your integrations list.
    </Check>
  </Step>
</Steps>

## Features

### Custom Domains

Use your Dub.co custom domains in Ordinal:
- Links use your branded domain
- Consistent branding across platforms
- Full control over link appearance

### Enhanced Analytics

Access Dub.co's advanced analytics:
- Detailed click tracking
- Geographic insights
- Device and browser data
- Referrer tracking

### Link Management

Manage links created in Ordinal:
- Links sync to your Dub.co dashboard
- Edit destinations anytime without changing the short link
- Bulk link management

## Configuration

### Default Settings

<Steps>
  <Step title="Open Dub.co Settings">
    Go to **Settings** → **Integrations** → **Dub.co**.
  </Step>
  <Step title="Set as Default">
    Enable **Use as default shortener** to use Dub.co for all new links.
  </Step>
  <Step title="Select Domain">
    Choose your default custom domain from Dub.co.
    
    <Tip>
    Select your primary brand domain to ensure consistent link appearance across all posts.
    </Tip>
  </Step>
  <Step title="Save">
    Click **Save** to apply settings.
    
    <Check>
    All new shortlinks will now use Dub.co with your custom domain.
    </Check>
  </Step>
</Steps>

### UTM Defaults

Configure default UTM parameters:
- Source: automatically set to platform
- Medium: social
- Campaign: configurable per post

## Troubleshooting

<AccordionGroup>
  <Accordion title="API key not working">
    Verify your API key is correct and has the necessary permissions. Generate a new key if needed.
  </Accordion>
  <Accordion title="Custom domain not appearing">
    Ensure your custom domain is verified and active in Dub.co before it will appear in Ordinal.
  </Accordion>
  <Accordion title="Links not syncing">
    Check your Dub.co workspace settings. Links sync automatically but may take a few minutes to appear.
  </Accordion>
</AccordionGroup>
